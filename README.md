# 🚀 AI智能财务管理系统

> **基于小企业会计准则的 SaaS 财务管理系统 · 集成 Deepseek AI 智能助手**

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🧠 **AI 智能助手** | 基于 Deepseek 大模型，支持自然语言录入凭证、智能校验科目、自动分析报表 |
| 📋 **完整凭证生命周期** | 录入 → 审核 → 过账 → 结转 → 结账，全流程闭环管理 |
| 🏢 **多公司账套** | 支持多公司独立核算，超级管理员统一管控 |
| 📊 **三大标准报表** | 资产负债表、利润表、科目汇总表，支持月/季/半年/年报 |
| 🔄 **智能期末处理** | 损益结转、月末结账自动完成，结账时自动缓存报表 |
| 🏦 **银行回单 & 发票** | PDF 导入自动解析，一键生成凭证 |
| 📱 **响应式设计** | 桌面端、平板、手机均可流畅使用 |
| 🔐 **角色权限** | 录入员、审核员、查看者、公司管理员、超级管理员 五级权限 |
| 🤖 **AI 自动操作** | 说"报销差旅费1200元"→ AI 自动创建凭证，无需手动填写 |
| 💰 **点数计费** | AI 功能按 Token 消耗点数，充值后使用，成本可控 |

---

## 🖥️ 技术栈

```
后端: Python 3.11 + FastAPI + SQLAlchemy + SQLite
前端: Jinja2 + Bootstrap 5 + Chart.js + Font Awesome 6
AI:   Deepseek API (OpenAI 兼容接口)
部署: 一键启动，零配置
```

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/yourusername/ai-finance.git
cd ai-finance
```

### 2. 配置 AI 密钥（可选）

```bash
# 复制配置模板
cp .env.example .env

# 编辑 .env 填入你的 Deepseek API Key
# 前往 https://platform.deepseek.com 注册获取
```

> 不配置 API Key 也可以使用系统的全部财务功能，只是 AI 助手功能不可用。

### 3. 一键启动

```bash
pip install -r requirements.txt
python run.py
```

浏览器访问 `http://localhost:8000`

### 4. 默认账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 🔑 超级管理员 | `admin` | `admin123` |
| 🏢 演示公司 | `jshyj` | `jshyj123` |

> **首次登录后请立即修改密码！**

## 🎯 功能速览

### 📝 凭证管理

凭证录入 → 提交审核 → 审核通过 → 凭证过账，全流程支持。

**AI 加持：** 直接对 AI 说"支付房租5000元"或"报销差旅费1200元"，AI 自动创建完整凭证。

### 📊 财务报表

| 报表类型 | 支持期间 | 导出格式 |
|----------|---------|---------|
| 资产负债表 | 月/季/半年/年 | Excel / PDF |
| 利润表 | 月/季/半年/年 | Excel / PDF |
| 科目汇总表 | 月/季/半年/年 | Excel / PDF |

### 🔄 期末处理

```
凭证过账 → 损益结转 → 月末结账
```

结账时自动缓存三大报表，反结账可解锁修改。

### 🤖 AI 智能助手

浮动图标常驻页面右下角，随时随地：

- **录入凭证：** "报销办公用品350元"
- **查询报表：** "查看上个月利润表"
- **科目校验：** "检查这个分录对不对"
- **财务知识：** "固定资产折旧怎么算"
- **功能指引：** "怎么导入银行回单"
